1. Download the Zip Folder "SE371ProjectSUBMIT.zip" and extract on users
2. Open cmd terminal
3. Run this commad to enter the project file --> cd SE371ProjectSUBMIT
3. Run this command if you don't have Node.js installed --> npm install
4. Wait for installation to finish
5. Run the project: npm run start
5. Visit http://localhost:3001

Visit http://localhost:3001/admin_panel
